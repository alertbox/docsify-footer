# Configure

Every Docsify site contains a `$docsify` configuration. You can also configure the plugin by modifying it:

### `loadFooter`

- Type: `Boolean|String`
- Default: `false`

Loads footer from the Markfown file `_footer.md` if set to `true`, otherwise from the path specified.
